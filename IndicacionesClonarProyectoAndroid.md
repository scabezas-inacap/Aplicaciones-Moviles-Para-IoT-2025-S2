# Clonar proyecto Android

1\. Ingresa a una carpeta segura en tu equipo para poder clonar el repositorio del curso. Por ejemplo puedes hacerlo desde Documentos.

2\. Accede a `git bash` y entra a la carpeta Documentos

```bash
cd Documents/
```

3\. Clona el repositorio principal de clases.

```bash
git clone https://github.com/scabezas-inacap/Aplicaciones-Moviles-Para-IoT-2025-S2.git
```

4\. Abre tu `Android Studio`.

5\. Abre el proyecto `MiAppFirebase` de la carpeta `Documentos`/`Aplicaciones-Moviles-Para-IoT-2025-S2`/`MiAppFirebase`

6\. Descarga desde la `Configuraci√≥n` de Firebase [https://console.firebase.google.cl](https://console.firebase.google.com/) el archivo `google-services.json`.

7\. Mueve el archivo `google-services.json` a la carpeta `app` de tu proyecto en `Android Studio`.
